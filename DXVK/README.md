# DXVK

Direct3D 8/9/10/11 to Vulkan translation layer.

| Variant | Target | Release tag |
| :--- | :--- | :--- |
| DXVK GPLAsync | x86 / x86_64 guests | [`Stable-DXVK`](https://github.com/WinNative-Emu/Components/releases/tag/Stable-DXVK) |
| DXVK ARM64EC | ARM64EC guests | [`Stable-DXVK-arm64ec`](https://github.com/WinNative-Emu/Components/releases/tag/Stable-DXVK-arm64ec) |
| DXVK-Sarek | GPUs without Vulkan 1.3 | [`Stable-DXVK-Sarek`](https://github.com/WinNative-Emu/Components/releases/tag/Stable-DXVK-Sarek) |

`.wcp` packages live on the GitHub Releases above — they are not committed to the
repository. This folder is the local staging area used when publishing them.
