| Variant | Target | Release tag |
| :--- | :--- | :--- |
| VKD3D-Proton | x86 / x86_64 guests | [`Stable-VKD3D`](https://github.com/WinNative-Emu/Components/releases/tag/Stable-VKD3D) |
