# FEXCore

FEX-Emu x86 / x86_64 emulation core, packaged as `libwow64fex.dll` and
`libarm64ecfex.dll` for ARM64EC guests.

| Variant | Release tag |
| :--- | :--- |
| FEXCore | [`Stable-FEXCore`](https://github.com/WinNative-Emu/Components/releases/tag/Stable-FEXCore) |

`.wcp` packages live on the GitHub Release above — they are not committed to the
repository. This folder is the local staging area used when publishing them.

Upstream: [FEX-Emu/FEX](https://github.com/FEX-Emu/FEX).
