# Wine

Windows compatibility layer, packaged as installable x86_64 Windows layers.

| Variant | Release tag |
| :--- | :--- |
| Wine (x86_64 layers) | [`Stable-Wine`](https://github.com/WinNative-Emu/Components/releases/tag/Stable-Wine) |

These layers are large (55–210 MB each), so they are hosted **only** as GitHub
Release assets — never committed to git. This folder is the local staging area
used when publishing them.

Upstream: [WineHQ](https://www.winehq.org/) ·
`ref4ik` builds via [REF4IK](https://github.com/REF4IK).
