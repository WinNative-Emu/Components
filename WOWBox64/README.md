# WOWBox64

WoW64 thunking layer that lets an ARM64 Windows environment run x86 applications.
Built from the Box64 source tree and used together with a matching Wine/Proton layer.

| Variant | Release tag |
| :--- | :--- |
| WOWBox64 | [`Stable-WOWBox64`](https://github.com/WinNative-Emu/Components/releases/tag/Stable-WOWBox64) |

`.wcp` packages live on the GitHub Release above — they are not committed to the
repository. This folder is the local staging area used when publishing them.

Upstream: [ptitSeb/box64](https://github.com/ptitSeb/box64).
