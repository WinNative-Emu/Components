# Proton

Valve's Wine fork with gaming patches, packaged as installable Windows layers.

| Variant | Release tag |
| :--- | :--- |
| Proton (ARM64EC / x86_64 layers) | [`Stable-Proton`](https://github.com/WinNative-Emu/Components/releases/tag/Stable-Proton) |

These layers are large (50–210 MB each), so they are hosted **only** as GitHub
Release assets — never committed to git. This folder is the local staging area
used when publishing them.

Upstream: [ValveSoftware/Proton](https://github.com/ValveSoftware/Proton) ·
ARM64EC layers via [GameNative](https://github.com/utkarshdalal/GameNative)
and the Winlator community.
